# Filament Table Filter Presets

## When to use this skill

Use this skill when working with FilamentTableFilterPresets features and when adding filters to a Filament resource.

## Features

- Feature 1: Set up the QueryBuilder constraints for the given resource.
- Feature 2: Add the `Wotz\FilamentTableFilterPresets\Concerns\HasSavedTableFilters` trait to the List page of the given resource.
